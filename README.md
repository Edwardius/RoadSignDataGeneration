# RoadSignDataGerenation
WATonomous is looking into simple synthetic data generation for road sign detection.

## Setup
Download the coco dataset into wato-gen
```
edwardius@linux-dev:~/code/RoadSignDataGeneration/wato_gen$ ./get_coco_dataset.sh
```
Ensure that the wato_road_sign_data folder is empty, new images will be generated here.
